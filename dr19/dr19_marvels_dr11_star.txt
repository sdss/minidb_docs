Summary
-------

Contains data for a MARVELS star (DR11 release).


Columns
-------

starname - The primary name of the star (STARNAME) []
twomass_name - 2MASS Star Catalog Name []
plate - Plate Name []
gsc_name - Guide Star Catalog Name []
tyc_name - Tycho Star Catalog Name []
hip_name - Hipparcos Star Catalog Name []
ra_final - Star Right Ascension [deg]
dec_final - Star Declination [deg]
gsc_b - GSC B Magnitude [mag]
gsc_v - GSC V Magnitude [mag]
twomass_j - 2MASS J Magnitude [mag]
twomass_h - 2MASS H Magnitude [mag]
twomass_k - 2MASS K Magnitude [mag]
sp1 - Hipparcos Spectral type 1 []
sp2 - Hipparcos Spectral type 2 []
rpm_log_g - Luminosity Class from SSPP* (Mainseq/Giant/Refstar) Refstar means it is a known planet host (different method used for year34 - see docs) []
teff - SSPP* effective temperature (different method used for year34 - see docs) [K]
log_g - SSPP* Surface gravity (different method used for year34 - see docs) []
feh - SSPP* Metallicity (different method used for year34 - see docs) []
gsc_b_e - Error in GSC B Magnitude [mag]
gsc_v_e - Error in GSC V Magnitude [mag]
twomass_j_e - Error in 2MASS J Magnitude [mag]
twomass_h_e - Error in 2MASS H Magnitude [mag]
twomass_k_e - Error in 2MASS H Magnitude [mag]
teff_e - Error in SSPP* effective temperature [K]
log_g_e - Error in SSPP* Surface gravity (different method used for year34 - see docs) (different method used for year34 - see docs) []
feh_e - Error in SSPP* Metallicity (different method used for year34 - see docs) []
epoch_0 - Epoch 0 []
ra_0 - Right Ascension at Epoch 0 [deg]
dec_0 - Declination at Epoch 0 [deg]
ra_twomass - 2MASS Right Ascension [deg]
dec_twomass - 2MASS Declination [deg]
gsc_pm_ra - GSC Proper Motion in RA pmra * cos(dec) [mas/yr]
gsc_pm_dec - GSC Proper MOtion in DEC [mas/yr]
gsc_pm_ra_e - Error in GSC Proper Motion in RA pmra * cos(dec) [mas/yr]
gsc_pm_dec_e - Error in GSC Proper MOtion in DEC [mas/yr]
tyc_b - Tycho B Magnitude [mag]
tyc_b_e - Error in Tycho B Magnitude [mag]
tyc_v - Tycho V Magnitude [mag]
tyc_v_e - Error in Tycho V Magnitude [mag]
hip_plx - Hipparcos Parallax [mas]
hip_plx_e - Error in Hipparcos Parallax [mas]
hip_sptype - Hipparcos Spectral Type []
twomass_designation - 2MASS Designation ID in dr19_twomass_psc []
tycho2_designation - Tycho2 Designation ID in dr19_tycho2 []
