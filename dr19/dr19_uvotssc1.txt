Summary
-------

Version 1.1 of the Swift UVOT Serendipitous Source Catalogue (UVOTSSC).

For full details, please see Page et al. (2015) at https://pos.sissa.it/233/037 and the online documentation at https://archive.stsci.edu/prepds/uvotssc/



Columns
-------

name - UVOTSSC1 name (JHHMMSS.s+DDMMSSa) []

obsid - Unique Swift observation ID []
nf - Number of filters included in this observation []
srcid - Unique source number []
radeg - Right ascension [degrees]
dedeg - Declination [degrees]
