Summary
-------

The table contains the list of cartons along with the target selection plan that generated them.


Columns
-------

carton_pk - The primary key. A sequential identifier.
carton - The name of the carton.
mapper_pk - The primary key of the mapper leading this carton. See ddr20_mapper.
category_pk - The primary key of the category in the ddr20_category table.
version_pk - The primary key of the version in the ddr20_version table.
program - The program name.
target_selection_plan - The target selection plan version for which this carton was run.
