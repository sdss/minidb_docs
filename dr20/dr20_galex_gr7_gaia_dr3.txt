Summary
-------

Cross-match between the Gaia DR3 catalogue and the GALEX General Release 7 catalogue. For more details about the GALEX columns refer to https://archive.stsci.edu/prepds/gcat/gcat_dataproducts.html.


Columns
-------

gaia_edr3_source_id - Gaia DR3 source identifier. Matches the source_id column in the dr20_gaia_dr3_source table.
galex_objid - This is the 64-bit integer version of the global object ID uniquely identifying this source including the tile number, band, product number, and other parameters, in the Gaia General Release 7 catalogue.
fuv_mag - Calibrated FUV magnitude on the AB system measured within the NUV-defined "AUTO" aperture. Sources with either a negative FLUX_FUV or without FUV observations will be flagged with values of either +99 or -99. [AB mag]
fuv_magerr - The error in MAG_FUV. Sources with either a negative flux or without FUV observations will be flagged with values of either +99 or -99. [AB mag]
nuv_mag - Calibrated NUV magnitude on the AB system. This is currently always set to the SExtractor "AUTO" magnitude. [AB mag]
nuv_magerr - The error in nuv_mag. [AB mag]
galex_separation - Separation between the Gaia DR3 source and the GALEX GR7 source. [arcsec]
pk - Unique identifier for each row in the table.
