Summary
-------

Chandra Source Catalog (CSC) 2.1 Optical/IR Counterparts Catalog
2022 Dec. See https://asc.harvard.edu/csc2.1/index.html for the CSC 2.1 release notes and documentation.


Columns
-------

csc21p_id - CSC 2.1 source name in the form "2CXO Jhhmmss.s{+|-}ddmmss"
csc21p_ensemble - CSC 2.1 ensemble. An ensemble is a set of overlapping stacks. It represents all the data used to match detections (i.e. to identify sources), and for calculating source properties.
ra - Preferred counterpart RA coordinate [deg]
dec - Preferred counterpart Dec coordinate [deg]
best_oir_cat - Best optical counterpart match; 'gdr3', 'lsdr10', 'ps1dr2', or '2mass' in that order of decreasing priority.
best_mag - Preferred magnitude of the optical counterpart [mag]
mag_type - Source of best_mag; GAIA_G, LS_g,LS_r, LS_z, PS_gMeanPSFMag, PS_rMeanPSFMag, PS_iMeanPSFMag, PS_zMeanPSFMag,2MASS_h_m in that order of decreasing priority.
gaia_dr3_srcid - Gaia DR3 source ID
ls_dr10_lsid - Legacy Survey DR10 ID
ls_dr10_objid - Legacy Survey DR10 object ID
ls_dr10_brickid - Legacy Survey DR10 brick ID
ls_dr10_release - Legacy Survey DR10 release
ps21p_objid - Pan-STARRS1 DR2 IPP unique object identifier
ps21p_ippobjid - Pan-STARRS1 DR2 IPP internal object identifier
tmass_designation - 2MASS designation
tmass_ra - RA from 2MASS [deg]
tmass_dec - Dec from 2MASS [deg]
tmass_h - H-band magnitude from 2MASS [mag]
csc21p_ra - RA of X-ray source
csc21p_dec - DEC of X-ray source
logfx - log10 of X-ray flux in xband
xband - X-ray bandpass for logfx, priority b,m,s,h,w
extent - X-ray extent flag
xsig - X-ray significance
xpriority - X-ray priority, calculated from significance
pk - Primary key identifier for this table row
